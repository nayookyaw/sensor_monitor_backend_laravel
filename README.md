# dependencies
1. Composer version 2.5.5
2. PHP 8.3.6
3. Laravel v11.3.1

Note - I used herd laravel Herd to install dependencies <br>
https://herd.laravel.com/windows <br>

# how to run project
Navigate to project root folder and run below command <br>

`php artisan serve`
